# SPOTIFY-MOCK APP
A Spotify clone app using the Spotify API.  


## About The Project
A Spotify clone app using the Spotify API. Allowing the user to search for songs and click a link to listen to the song in a new tab.


## Built With

- Create-react-app
- HTML / CSS

## Prerequisites
 - Git
 - Node.js
 - NPM
